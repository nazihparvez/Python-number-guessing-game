# Python-number-guessing-game
Hello, so this simple project still took some time for me to make, but its worth it cuz i understand the basics a bit more now.

In this project, when you run the code, you have started the game!

How to play:
-enter the limit of the numbers, then
-enter the number of chances you want to give yourself, then
-keep guessing until your chances run out or until you win!
